# AI Assistant Guidance: Product Context

## Quick Reference for AI Work
- **Project Type**: Static ornamental fish showcase website
- **Primary Goal**: Display premium fish specimens with aquarium aesthetics
- **User Experience**: Elegant, professional showcase for aquarium enthusiasts
- **Key Differentiator**: Glass morphism UI with bubble animations

## AI Assistant Focus Areas
1. **Maintain visual consistency**: Dark theme, glass effects, aquarium aesthetic
2. **Preserve performance**: Keep site static-first, optimize images
3. **Follow data patterns**: Consistent JSON structure for fish catalog
4. **Enhance user experience**: Smooth animations, responsive design

## What to Avoid
- Breaking the dark theme aesthetic
- Adding unnecessary dependencies
- Complex build processes (keep it static)
- Database requirements (file-based storage only)
